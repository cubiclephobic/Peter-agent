# personal-tone

Receive a raw draft and original message from /draft-writer. 

Pull 10–15 messages Corinna has already sent from Gmail sent folder (via Gmail MCP). Analyze writing patterns: sentence length, vocabulary level, punctuation habits, how she opens and closes messages, use of em-dashes (she avoids them), paragraph length, tone warmth.

Rewrite the draft to match those patterns exactly. Do not change the meaning or add new content. Apply Zaradigm voice rules: no em-dashes, no rhetorical questions as hooks, no banned words (delve, tapestry, pivotal, showcase, etc.), short paragraphs, punchy sentences.

After rewriting, immediately invoke the /qa-check skill with the rewritten message and the original message for reference.
