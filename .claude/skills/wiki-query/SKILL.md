# wiki-query

Take a question as input. Scan wiki/index.md to identify the 1–3 topic files most likely to contain a relevant answer. For each match, return the file name and a one-sentence reason why it's relevant.

**Do not answer the question.** Only surface where to look.

If no topic files are clearly relevant, say so.

Return only file names and relevance reasons. Never answer the question directly.
