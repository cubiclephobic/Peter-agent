## Website Update Process (Horizons → hPanel)

1. In Horizons: Code tab → Export code → save zip → extract
2. Right-click extracted folder → Open in Terminal
3. `npm install` (first time only)
4. `npm run build` — wait for ✓
5. `open dist`
6. In hPanel File Manager → public_html → upload/replace `index.html` + entire `assets` folder
7. Flush cache in hPanel → verify at mediumseagreen-sandpiper-985735.hostingersite.com → Cmd+Shift+R

**Note:** Always upload both index.html AND assets together — they must match.
