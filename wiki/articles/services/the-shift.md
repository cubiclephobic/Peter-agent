## Google Slides / Apps Script — Technical Reference

**Apps Script project:** https://script.google.com/u/0/home/projects/1KiaJfOc5KtnxJ6zN3oLhp8j5PDMxk_HdQ66SwPJgIFgS0jJC6GvBL2Sl/edit

**Presenter notes formatting specs:**
- "The Question:" = bold heading; question text indented (indentStart = 30)
- "Hidden Layers:" = bold heading; bullets use native Google Slides list (DISC_CIRCLE_SQUARE preset), indentStart=36, indentFirstLine=18, spaceAbove=12, spaceBelow=12
- Bullet lead-ins are italic; followed by regular text
- "Possible Framework Tie-In:" = bold heading with non-bold text on same paragraph
- No empty paragraphs between sections

**Common issues when adding new slides:**
- indentStart defaults to 0
- Empty paragraphs get inserted
- Literal "• " Unicode characters appear instead of native list formatting

**Fixes:**
- Use browser console injection to paste code into Apps Script Monaco editor: `window.monaco.editor.getModels()[0].setValue(\`...\`)`
- Loop indices are 0-based (Slide 9 in UI = index 8 in script)
- Always iterate backwards when deleting paragraphs; re-read paras after any deletion
